# use step

1. ### move qcd and qcd.conf to /usr/local/bin   
>`sudo mv qcd* /usr/local/bin`

2. ### add alias    
>`alias qcd='. qcd'`    

 or you can edit ~/.bashrc       
>`echo "alias qcd='. qcd'" >> ~/.bashrc`    

3. ### edit qcd.conf     
>example:    
>my=/home/testuser01

4. ### add path
> - jump to path
> - `j add ${NAME}`
